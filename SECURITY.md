# Security Policy

Feather is preview software. Please do not report security-sensitive issues in
public GitHub issues.

If you believe you have found a vulnerability, email the maintainers or use the
private vulnerability reporting channel on the project repository once it is
enabled. Include:

- Affected version or commit.
- Operating system, GPU backend, and driver details when relevant.
- A minimal reproduction or proof of concept.
- Whether the issue requires untrusted shader/kernel input, untrusted files, or
  a malicious native library.

The maintainers will acknowledge the report, investigate scope, and coordinate
a fix before public disclosure when appropriate.
